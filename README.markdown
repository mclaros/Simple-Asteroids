#Simple Asteroids

##Instructions

 * UP, LEFT, RIGHT - to move
 * SPACEBAR - to shoot

 ##Usage

 Just open the `index.html` file on any modern browser.
 __NOTE:__ This simple game has _no_ winning conditions! It only ends if you lose (get hit by an asteroid), or you close the page.
